Auto Tok — Privacy Policy

Last updated: April 5, 2026

Auto Tok is a personal development application. This policy describes how data is handled:

  • Data collected: This application accesses OAuth tokens for YouTube and TikTok to enable video publishing on your behalf. No personal data is collected or stored on external servers.
  • Data storage: OAuth tokens are stored locally on the developer's machine only.
  • Third-party services: This application uses YouTube Data API v3 and TikTok Content Posting API. Their respective privacy policies apply.
  • Data sharing: No user data is shared with third parties beyond what is required by the platform APIs.

For questions, contact the developer directly.
